# AI-генератор экзаменационных вопросов (MVP)

Стартовая реализация платформы по ТЗ: контекст генерации, темы, матрица количества вопросов, два режима, прогресс, результаты по языкам.

## Что уже реализовано

- UI шагов 1–5 из user flow:
  - настройка контекста (факультет, курс, тип экзамена, предмет, языки, режим);
  - добавление тем (вручную и списком);
  - таблица количества `easy/medium/hard` с `min=0`, `max=100`;
  - запуск/отмена генерации;
  - просмотр результатов по вкладкам `RU / TJ / EN`, кнопка копирования.
- Backend API каркас:
  - `POST /api/generate/start`
  - `GET /api/generate/:jobId/status`
  - `POST /api/generate/:jobId/cancel`
  - `GET /api/generate/:jobId/result`
  - `POST /api/export/docx` (пока заглушка)
- Хранение данных в памяти процесса (MVP без БД).
- Мок-генератор вопросов для end-to-end теста UX.

## Запуск

```bash
npm start
```

После запуска откройте:

```text
http://localhost:3000
```
